# KNOWN ISSUES / INSTALL NOTES
#
# 1. MIDIUtil package name on PyPI is "MIDIUtil" (capital M, U):
#       pip install MIDIUtil
#    NOT "midiutil" (lowercase) — that is a different/older package.
#
# 2. Minimal install (notation + parsing, no MIDI/audio/AI):
#       pip install raganotate
#    Core has zero third-party dependencies — pure Python stdlib.
#
# 3. For real-time audio playback (in addition to file export):
#       pip install pyaudio        # cross-platform real-time playback
#       pip install simpleaudio    # simpler alternative (Windows/Mac/Linux)
#
# 4. transformers + torch are large (~2 GB). Skip if not training models:
#       pip install raganotate[ai]   # installs datasets + tokenizers only
#
# 5. On Windows, soundfile may need:
#       pip install soundfile --pre
#
# 6. Full dev setup from repo root:
#       cd packages/python
#       pip install -e ".[all]"
#       pytest -v
